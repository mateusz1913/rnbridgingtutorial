#### Source files stubs

Finally, let's create stubs for source files. For JS, create following empty files:

- `src/index.ts`
- `src/NativeScreenOrientationModule.ts`

- `ios/ScreenOrientationModuleImpl.swift`
- `ios/ScreenOrientationPackage-Bridging-Header.h`

</TabItem>
<TabItem value="objc" label="ObjC++ only">
