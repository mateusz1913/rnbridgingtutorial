#### Source files stubs

Finally, let's create stubs for source files. For JS, create following empty files:

- `src/index.ts`
- `src/NativeSaveFilePickerModule.ts`

For iOS implementation, create:

<Tabs groupId="ios_lang">
<TabItem value="swift" label="ObjC++ & Swift">

- `ios/SaveFilePickerModule.h`
- `ios/SaveFilePickerModule.mm`
- `ios/SaveFilePickerModuleImpl.swift`
- `ios/SaveFilePickerPackage-Bridging-Header.h`

</TabItem>
<TabItem value="objc" label="ObjC++ only">

- `ios/SaveFilePickerModule.h`
- `ios/SaveFilePickerModule.mm`
- `ios/SaveFilePickerModuleImpl.h`
- `ios/SaveFilePickerModuleImpl.mm`

</TabItem>
</Tabs>
