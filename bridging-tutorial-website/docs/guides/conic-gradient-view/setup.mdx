#### Source files stubs

Finally, let's create stubs for source files. For JS, create following empty files:

- `src/index.ts`
- `src/ConicGradientView.tsx`
- `src/ConicGradientViewNativeComponent.ts`

For iOS implementation, create:

- `ios/ConicGradientView.h`
- `ios/ConicGradientView.mm`
- `ios/ConicGradientViewComponentView.h`
- `ios/ConicGradientViewComponentView.mm`
- `ios/ConicGradientViewManager.h`
- `ios/ConicGradientViewManager.mm`

:::info

Extending `<View />` on iOS can be done only with Objective-C

:::
