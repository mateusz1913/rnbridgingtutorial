#### Add native libraries as dependencies for the package

When developing some Android native code, you often need to use some external package, whether it's from Jetpack, MaterialComponents or some 3rd party.
Usually those libraries are integrated with [Gradle](https://docs.gradle.org/current/userguide/userguide.html).

:::tip

For more information on how to add dependencies to Android project visit [Android's dedicated docs](https://developer.android.com/studio/build/dependencies).

:::

Each RN library that includes some Android native code is, in fact, integrated with Gradle and our native list will also need to depend on Jetpack libraries:

- [`CardView`](https://developer.android.com/develop/ui/views/layout/cardview)
- [`ConstraintLayout`](https://developer.android.com/develop/ui/views/layout/constraint-layout)
- [`RecyclerView`](https://developer.android.com/develop/ui/views/layout/recyclerview)

So let's not waste time and navigate to the `build.gradle` in our package.

<Tabs groupId="gradle">
<TabItem value="groovy" label="Groovy script (build.gradle)">

```groovy title="android/build.gradle"
buildscript {
    // ...
}

// ...

android {
    // ...
}

repositories {
    // ...
}

dependencies {
    // ...

// highlight-start
    // Add the dependency to the Jetpack libraries
    implementation "androidx.cardview:cardview:1.0.0"
    implementation "androidx.constraintlayout:constraintlayout:2.1.4"
    implementation "androidx.recyclerview:recyclerview:1.2.1"
// highlight-end
}
```

</TabItem>
<TabItem value="kotlin" label="Kotlin script (build.gradle.kts)">

```kotlin title="android/build.gradle.kts"
buildscript {
    // ...
}

// ...

plugins {
    // ...
}

// ...

android {
    // ...
}

repositories {
    // ...
}

// ...

dependencies {
    // ...

// highlight-start
    // Add the dependency to the Jetpack libraries
    implementation("androidx.cardview:cardview:1.0.0")
    implementation("androidx.constraintlayout:constraintlayout:2.1.4")
    implementation("androidx.recyclerview:recyclerview:1.2.1")
// highlight-end
}
```

</TabItem>
</Tabs>

To add a dependency we need to write `implementation` keyword and declare the package (and its version) we want to include.
After that you can invoke Gradle Sync in the Android Studio.

:::info

In this guide, version 1.0.0 of CardView, 2.1.4 of ConstraintLayout and 1.2.1 of RecyclerView are used. In your case these versions may be different, you can visit [Jetpack's releases page](https://developer.android.com/jetpack/androidx/versions) and check available versions.

:::
